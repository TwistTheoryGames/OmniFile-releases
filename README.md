## What is OmniFile?

OmniFile is a privacy-first desktop app that lets you find any file across your local drives and cloud storage with a single keyboard shortcut (**Ctrl+Space**). Powered by [Tantivy](https://github.com/quickwit-oss/tantivy), all indexing happens locally — your data never leaves your device.

## Key Features

| Feature | Description |
|---------|-------------|
| ⚡ **Instant Search** | Find files in milliseconds with fuzzy matching |
| ☁️ **Cloud Integration** | Search Google Drive, Dropbox, Box, SharePoint, Slack, Notion, and GitHub |
| 🔒 **Privacy-First** | All indexes stored locally. No data collection. Works offline |
| 🖥️ **Cross-Platform** | Native app for macOS and Windows |
| 📁 **Smart Workspaces** | Organize searches by project, folder, or file type |
| ⌨️ **Keyboard-Driven** | Open, reveal, copy paths — all without touching the mouse |
| 🚀 **Native Performance** | Built with Rust and Tauri for minimal resource usage |

## How It Works

1. **Install** — Download and install OmniFile
2. **Add folders** — Choose which directories to index
3. **Search** — Press **Ctrl+Space** to open the search bar and start typing
4. **Connect clouds** *(Pro)* — Link Google Drive, Dropbox, and more to search everything in one place
